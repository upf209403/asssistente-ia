# Estruturas de repetição

## Definição
- Estruturas ou laços de repetição, também chamados loops, são objetos da programação 

## Por que existem estruturas de repetição?
- Estruturas de repetição servem para executar uma sequência de passoas

- "While" significa "enquanto"
